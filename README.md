# market-ticker
🚀 Crypto Price Ticker (Python)
This is a lightweight Python-based desktop utility designed to monitor real-time cryptocurrency prices using the Binance API. It is built for traders and developers who need to stay updated on market movements without constantly checking an exchange.

✨ Features
Real-time Monitoring: Fetches live prices for BTC, ETH, and BNB.

Custom Desktop Notifications: Integrated alerts for price changes.

Lightweight: Minimal CPU and RAM usage.

Easy Setup: Built with Python and standard libraries.

🛠️ Built With
Language: Python 3.x

API: Binance Public API

Libraries: requests, tkinter (for UI), json

🚀 Getting Started
Prerequisites
Make sure you have Python installed on your system. You will also need the requests library:

Bash
pip install requests
Installation & Usage
Clone this repository:

Bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
Navigate to the project folder and run the script:

Bash
python main.py
